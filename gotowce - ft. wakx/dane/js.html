🧠 JavaScript – podstawy (INF.03)

  • JavaScript służy do dynamicznych zmian HTML/CSS
  • Kod w pliku .js lub w znaczniku <script>
  • Podpięcie pliku JS: <script src="skrypty.js"></script>
  • Konsola: F12 → Console

📦 Zmienne


let a = 10;        // najczęściej używane
const b = 5;      // stała
var c = 3;        // stary zapis

typeof a;         // sprawdza typ

➕ Operatory

  • Arytmetyczne: + - * / % ++ --
  • Porównania: == === != !== > >= < <=
  • Logiczne: && || !
  • Przypisania: = += -= *= /=

🔁 Pętle


for(let i=0;i<5;i++){
  console.log(i);
}

let arr = ["A","B","C"];
arr.forEach(el => {
  console.log(el);
});

📚 Tablice


let arr = [1,2,3];
arr.push(4);
arr.pop();

arr.forEach(el => {
  console.log(el);
});

🧱 Obiekty


let osoba = {
  imie: "Jan",
  wiek: 20
};

for(let k in osoba){
  console.log(k, osoba[k]);
}

🖱️ Manipulacja HTML


document.getElementById("id").innerHTML = "Nowy tekst";
document.getElementById("id").textContent = "Tylko tekst";

document.querySelector(".klasa");
document.querySelectorAll("li");

🎯 Zdarzenia


document.getElementById("btn").onclick = function(){
  alert("Klik!");
};

document.getElementById("btn")
  .addEventListener("click", function(){
    console.log("OK");
  });

🔀 Warunki


if(a > 5){
  console.log("Większe");
}else{
  console.log("Mniejsze");
}

switch(a){
  case 1: console.log("1"); break;
  default: console.log("inne");
}

📌 Typowe zadania INF.03

  • Zmiana treści po kliknięciu przycisku
  • Generowanie listy z tablicy
  • Prosty alert() lub console.log()

⚠️ Najczęstsze błędy

  • Brak id w getElementById
  • JS uruchamiany przed HTML
  • Literówki w nazwach zmiennych
  • innerHTML bez cudzysłowów

🚑 ULTRA SZYBKA WERSJA


document.getElementById("id").innerHTML = "tekst";

let arr = [1,2,3];
arr.forEach(e => console.log(e));

function f(x){
  return x*2;
}

✅ Wskazówki egzaminacyjne

  • Wystarczy czysty JS (bez frameworków)
  • forEach w zupełności wystarcza
  • alert() i console.log() do testów
  • Zawsze sprawdzaj id elementów
