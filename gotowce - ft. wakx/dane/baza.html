🧠 Schemat myślenia (NAJWAŻNIEJSZE)

  • ❌ HTML NIE łączy się z bazą danych
  • ✅ PHP łączy się z bazą MySQL
  • HTML tylko wyświetla dane
  • PHP pobiera dane i wstawia je do HTML
  • Dlatego plik na egzaminie ma rozszerzenie .php

📁 Struktura plików – klasyk INF.03

/polaczenie.php   (bardzo zalecane)

🗄️ Baza danych – phpMyAdmin

1️⃣ Tworzenie bazy

  • Nazwa: medica
  • Kodowanie: utf8mb4_general_ci

2️⃣ Tworzenie tabeli


CREATE TABLE abonamenty (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nazwa VARCHAR(50),
    cena INT
);

3️⃣ Dodawanie danych


INSERT INTO abonamenty (nazwa, cena) VALUES
('Standardowy', 200),
('Premium', 500),
('Dziecko', 150);

🔌 Połączenie z bazą – PHP

Plik: polaczenie.php


<?php
$conn = mysqli_connect("localhost", "root", "", "medica");

if (!$conn) {
    die("Błąd połączenia z bazą");
}
?>

📌 Dane na egzaminie

  • serwer: localhost
  • login: root
  • hasło: puste ""

🌐 index.php – PHP + HTML


<?php
require_once "polaczenie.php";

$zapytanie = "SELECT * FROM abonamenty";
$wynik = mysqli_query($conn, $zapytanie);
?>

<!DOCTYPE html>
<html lang="pl">
<head>
  <meta charset="UTF-8">
  <title>Przychodnia Medica</title>
  <link rel="stylesheet" href="styl.css">
</head>

<body>

<header>
  <h1>Abonamenty</h1>
</header>

<main>

<?php
while ($wiersz = mysqli_fetch_assoc($wynik)) {
    echo "<section>";
    echo "<h2>" . $wiersz['nazwa'] . "</h2>";
    echo "<p>Cena: " . $wiersz['cena'] . " zł</p>";
    echo "</section>";
}
?>

</main>
</body>
</html>

📌 Częste warianty zapytań SQL (EGZAMIN)


SELECT * FROM abonamenty;
SELECT nazwa FROM abonamenty;
SELECT * FROM abonamenty WHERE cena > 200;
SELECT * FROM abonamenty ORDER BY cena ASC;
SELECT * FROM abonamenty ORDER BY cena DESC;

⚠️ Najczęstsze błędy

  • ❌ index.html zamiast index.php
  • ❌ brak mysqli_query()
  • ❌ brak pętli while
  • ❌ literówki w nazwie bazy lub tabeli
  • ❌ brak AUTO_INCREMENT
  • ❌ brak <link rel="stylesheet">
  • ❌ brak UTF-8

✅ Wskazówki egzaminacyjne

  • Zawsze sprawdzaj nazwę bazy i tabeli

  • Najczęściej są 3 rekordy: Standardowy / Premium / Dziecko
  • Wszystko dynamiczne robisz w PHP
  • HTML to tylko szkielet
